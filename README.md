# MangaTR

MangaTR is a desktop application that automatically translates English manga into Turkish.

## Features (Planned)

- Long scroll manga reader
- OCR text detection
- Automatic English → Turkish translation
- Speech bubble cleaning
- Turkish text rendering
- Fast local processing
